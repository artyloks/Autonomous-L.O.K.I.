# 🚀 Autonomous-L.O.K.I. 

## 📖 Overview
Autonomous-L.O.K.I. (Logical Optimization & Knowledge Intelligence) is a 2026-edition AI Passive Income Flywheel. This system creates a self-sustaining loop where agentic AI entities collaborate to generate content, analyze market trends, and optimize monetization strategies.

### ⚙️ How it Works
1. **Agentic Collaboration:** Multiple specialized agents (WriterBot, MarketBot, LinkBot) hand off data to one another.
2. **Market Intelligence:** Live analysis of Stock and Crypto markets using `yfinance`.
3. **Self-Optimization:** A built-in memory system (`memory.json`) tracks performance and "rewires" AI prompts if failure thresholds are met.
4. **No API Keys:** Runs entirely on free, open-source models from Hugging Face.

## 🛠️ Setup & Usage
**Requirements:**
`pip install transformers requests beautifulsoup4 yfinance`

**Execution:**
Run `python main.py` to start the autonomous cycle. The system will execute one full loop and then sleep for 1 hour before self-improving for the next cycle.
